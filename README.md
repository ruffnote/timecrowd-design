# timecrowd-design

TimeCrowd design packages

## Bootstrap Theme

### Install

```
$ npm install -g grunt-cli
$ cd packages/timecrowd-bootstrap
$ npm install
$ gem install bundler
$ bundle install
```

### Development

```
$ grunt watch
```

## React Storybook

### Install

```
$ cd packages/timecrowd-stories
$ npm install
```

### Run

```
$ yarn run storybook
```
